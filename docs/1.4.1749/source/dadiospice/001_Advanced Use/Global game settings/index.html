Global Game Settings

This section deals with the Global Games Settings window.

When working with GameMaker: Studio on a new project, it is vitally important that one of the first things you do is set up the Global Games Settings. It is through these options that you define how your game final game will be created and what information will be stored along with it for the various possible target modules.

The settings can be changed by double clicking on Global Game Settings in the resource tree at the left of the screen, or by clicking the icon at the top of the IDE, or by selecting "Global Game Preferences" from the drop down resources menu. The window that opens is further subdivided into the following tabs:

General Settings

The following pages deal with the settings that affect multiple (or even all) target platforms:

  1. General Settings
  2. Texture Groups
  3. Audio Groups
  4. Project Info
  5. Source Control
  6. In App Purchases
  7. Analytics
  8. Advertising

Target Module Settings

Apart from the general settings for your game, each target module has a separate tab where you can set up the target-specific details:

  1. Windows
  2. Windows UWP
  3. Windows 8
  4. Windows Phone
  5. Mac OS
  6. Ubuntu (Linux)
  7. HTML5
  8. Android
  9. Amazon Fire
  10. iOS
  11. Tizen
  12. Steam API

Please note that the availability and functionality of these tabs will depend on the version of GameMaker: Studio that you are using. See here for more information.
